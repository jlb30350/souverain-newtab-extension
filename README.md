Installation (30 secondes)

1. Télécharger le fichier ZIP
2. Clic droit → Extraire ici
3. Ouvrir Chrome → chrome://extensions
4. Activer Mode développeur (en haut à droite)
5. Cliquer "Charger l’extension non empaquetée"
6. Sélectionner le dossier extrait
7. Ouvrir un nouvel onglet → terminé
